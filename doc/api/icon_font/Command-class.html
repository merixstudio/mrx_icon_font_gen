Command

Command class Null safety

An abstract drawing command.

SVG path is represented by a series of drawing commands, each consisting of a single letter name, and in most cases an unlimited lit of parameters.

Commands come in to variants:

  • Absolute (uppercase name) - indicates that the following parameters are to be treated as absolute coordinates.
  • Relative (lowercase name) - coordinates of these commands represent an offset from the current position of the "pen" (a point, where the preceding command finished drawing).
